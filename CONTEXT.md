# Gaura Style — Storefront

The gaura.style fine-jewelry storefront. This glossary pins language used across the
home page and its category surfaces so the same word means the same thing in markup,
copy, and design decisions.

## Language

**Category**:
A true product category a shopper can browse — Rings, Earrings, Bracelets, Pendants
(and Brooches). These are the only things that belong in a "category" row or tile.
_Avoid_: Collection (means something else, below), Section.

**Collection**:
A brand/merchandising grouping of products that is NOT a product category — e.g. a
seasonal or themed edit. "Shop the Collection" meant "shop everything," not "shop a
category." Do not use interchangeably with Category.
_Avoid_: Category.

**Merchandising bucket**:
A non-category filter surfaced like a category for promotion — e.g. "New arrivals" /
"New In". Looks like a category in the nav but is not one; excluded from the hero
category tiles for this reason.
_Avoid_: Category.

**Editorial link**:
A nav entry that leads to brand/content, not products — e.g. "World of Gaura". Never
belongs in a product-category row.
_Avoid_: Category.

**Hero category tiles**:
The pictured category choices placed in the home hero (over the video). The hero's
*only* call to action — there is no separate "shop all" CTA alongside them.

**Collection showcase**:
The editorial block directly below the hero: a contained, full-viewport-height
composition with a typographic headline on the left (~1/4 width) and an asymmetric
six-block puzzle grid of looping media on the right (~3/4). Each block surfaces a
ring **Collection** (above), NOT a Category — so it is distinct from the Hero
category tiles and never functions as a category row. Block labels are small white
plates with a gold accent line that draws in on hover (the one-voice gold rule).
_Avoid_: calling these blocks categories or tiles.
